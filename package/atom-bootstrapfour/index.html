Bootstrapfour

Install with: 
    apm install atom-bootstrapfour

Bootstrap 4 Support for Atom

Provides Bootstrap 4 classes and HTML tags with autocompletion features.

Install: bash apm install atom-bootstrap4

Key Features:

  • Supports autocompletion of all helper classes in JavaScript, CSS and HTML
  • Snippet support for all components.
  • Snippet support for templates.
  • Snippet support for layout.
  • Snippet support for content.
