W Atom Snippet by GeekWeier

Install with: 
    apm install w-atom-snippet

AtomSnippets

最近折腾编辑器，之前用的 sublime, 久了就烦啦，看到 atom 这颜值我喜欢。 平常都用 atom 写 react之类的开发，所以代码段肯定不能少的，也安装了好多受欢迎的，但是总觉的他们的太杂，而且有的也不怎么符合我的代码格式，所以就自己尝试着写一个自己用的。

支持

  • es6
  • reactjs
  • redux
  • react-router
  • react-native
  • js(部分常用的原生的 js)
