Description

A north-bluish dark theme based on the same named color palette designed by Arctic Ice Studio.
This is a darker and more blue-tinged variation.

Installation

Download and install the recommended font, read section Recommendations for more informations.

Install via apm: apm install northem-dark-atom-ui
Alternatively open Settings > Install, set the filter to Themes and search for northem-dark-atom-ui.

Activate it by going to the Settings > Themes section and select Northem Dark from the UI Theme drop-down menu.
