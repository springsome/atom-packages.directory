Dicom Dump

Install with: 
    apm install dicom-dump

DICOM viewer for Atom

Overview

This package provides a point-and-click interface for exploring the content of DICOM files. At the moment, the functionality provided is allowing to run DCMTK command-line tools dcmdump and dsrdump.

By using this Atom package instead of running the DCMTK tools in the command line, you can:

  • conveniently copy content to the clipboard
  • do folding of the indented sections of the dump
  • search for content of interest
  • add a lot more features using powerful Atom engine ;)

Usage

  1. Install the package from Atom > Settings > Install
  2. Download and install DCMTK binaries for your system from http://dicom.offis.de/dcmtk.php.en (HINT: binaries of DCMTK command line tools are build and packaged with 3D Slicer (http://slicer.org) - you can install Slicer and point atom-dicom-dump to the subdirectory of 3D Slicer installation where dcmdump is located! For example, on Mac OS X it is /Applications/Slicer.app/Contents/bin)
  3. Set the location of the DCMTK executables in the dicom-dump settings
  4. After opening a DICOM file in Atom, use context meny to invoke dcmdump or dsrdump tools.
