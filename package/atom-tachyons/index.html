Tachyons

Install with: 
    apm install atom-tachyons

Tachyon Support for Atom

Provides Tachyon classes with autocompletion and component snippets.

Install:

bash apm install atom-tachyons

Key Features:

  • Supports autocompletion of all helper classes in JavaScript, CSS and HTML
  • Snippet support for all components in js, jsx, and html files.

Components

All components also have autocompletion support for extra options.
