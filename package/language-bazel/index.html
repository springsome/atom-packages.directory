Bazel

Install with: 
    apm install language-bazel

Bazel language support for atom.

Adds syntax highlighting to Googleâ€™s Bazel language from bazel.io.

Features

  • Syntax Highlighting
  • Snippets

Snippets

``` Java: jb -> java_binary jl -> java_library jt -> java_test ji -> java_import jp -> java_plugin

Android: ab -> android_binary al -> android_library

Python: pb -> py_binary pl -> py_library pt -> py_test

C / C++: cb -> cc_binary cl -> cc_library ct -> cc_test

Shell: sb -> sh_binary sl -> sh_library st -> sh_test

General: gr -> genrule fg -> filegroup ts -> test_suite ```
