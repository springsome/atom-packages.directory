Pinned tabs for Atom

A simple package for the Atom text editor from GitHub that allows you to pin tabs. Inspired by the pin tab feature from Google Chrome.

For the best experience of this package, I recommend using it with the file-icons package.

Usage

There are three ways to pin/unpin a tab using this package. - Via the context menu of a tab. - Via the keyboard shortcut ctrl-alt-p. - Via the command-palette, by typing pin.

Installation

Via the Atom Package Manager (APM) $ apm install pinned-tabs

Or via Git clone $ cd ~/.atom/packages $ git clone https://github.com/ericcornelissen/pinned-tabs-for-atom --depth=1
