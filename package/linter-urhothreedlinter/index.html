linter-urho3dlinter

Fork of linter-gcc

Linter plugin for Linter, provides an interface to Urho3DLinter Angelscript/Lua

Similarly to linter-gcc, a local config file will be searched for named .urho-flags.json, example: json { "enabled": true, "execPath": "./Urho3DLinter", "defaultFlags": "-log warning", "defaultPackages": "./Data;./CoreData" }
