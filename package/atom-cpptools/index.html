Cpptools

Install with: 
    apm install atom-cpptools

C/C++ for Atom editor

Simple tools compile C/C++ for Atom editor.

Install

Using apm: sh apm install atom-cpptools

Or search for atom-cpptools in Atom settings view.

Node: You will need install gcc/g++. * If you using Windows you can download and setup TDM-GCC. * For Ubuntu users, you can setup gcc/g++ though Terminal: apt-get install g++ * For Mac users, Xcode is not a prerequisite but you need to at least install the Xcode Command Line Tools. This site has a more detailed walkthrough.
