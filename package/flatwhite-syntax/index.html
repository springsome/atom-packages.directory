Flatwhite Syntax

Install with: 
    apm install flatwhite-syntax

Minimal light syntax theme. 5 colored backgrounds used for code highlighting.

Easy on the eyes. Maintain 6:1 contrast between text and background colors. WCAG 2.0 on contrast ratio

Supported languages

  • C
  • Clojure
  • CSS / Less / SASS / SCSS
  • Elixir (with language-elixir package)
  • GitHub Markdown
  • Go
  • Haskell (with language-haskell)
  • HTML
  • Jade
  • JavaScript (better with language-babel) / CoffeeScript
  • React JSX (with language-babel)
  • Ruby / RoR / ERB / Slim (with slim-language package)
  • Rust (with language-rust)
  • PHP
  • Python

If you need support for specific language, please, open an issue.

Settings

You can choose between 2 embedded highlighting variants: simple (default) and full.
