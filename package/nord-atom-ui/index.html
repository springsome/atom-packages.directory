A arctic, north-bluish clean and elegant minimal Atom UI.

Designed for a optimized fluent and clear workflow.
Based on the Nord color palette.
Fits the nord-atom-syntax theme.

> Icons provided by file-icons, minimap by minimap.

Getting started

### Installation <img src=”https://atom.io/favicon.ico” width=16 height=16/> apm
shell apm install nord-atom-ui

<img src=”https://atom.io/favicon.ico” width=16 height=16/> Atom
1. Open Settings > Install 2. Set the filter to Themes 3. Search for nord-atom-ui and install

Activation 1. Open Settings > Themes 2. Select Nord Atom from the UI Theme drop-down menu

Features

> Floating modal overlays with a big font type.

> Smooth, non-obtrusive transitions for UI components.

> Optimized tab layout by using the full editor width.

> Flat, colored dirty state indicator.
