Note that this package is still WIP, see v1 roadmap milestone for details. See the change log for whatâ€™s new.

Your mental notes at your fingertips!

For those of you who used Notational Velocity or nvalt before should feel right at home.

Usage

The package is lazy-loaded, use the Textual Velocity: Start Session command to get started.

By default notes are saved in ~/.atom/notes, you can change it through the textual-velocity:path config setting.

You may also want to set a global shortcut to toggle the Atom window, see my init config for an example of how this can be setup when the package is activated.

The core.ignoredNames and core.excludeVcsIgnoredPaths are respected, but you need to restart a session for the changes to apply.

See keymaps for available shortcuts.

Also, I recommend the following packages, that works great in combination with textual-velocity: - quick-file-actions - Quickly copy, delete, move, and create new files - file-icons - Assign file extension icons and colours for improved visual grepping - preview - Ultimate previewer of source code in Atom. - recent-files-fuzzy-finder - Find recently opened files easier - block-travel - Quickly travel up/down between blocks of code

Installation

apm install textual-velocity, or search & install it through the settings.
