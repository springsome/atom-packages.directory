North-bluish dark theme based on the same named color palette Northem Dark.

Getting started

### Installation <img src=”https://atom.io/favicon.ico” width=16 height=16/> apm
shell apm install northem-dark-atom-syntax

<img src=”https://atom.io/favicon.ico” width=16 height=16/> Atom
1. Open Settings > Install 2. Set the filter to Themes 3. Search for northem-dark-atom-syntax and install

Activation 1. Open Settings > Themes 2. Select Northem Dark Atom from the Syntax Theme drop-down menu
